# Controls

```markdown
Mouse look to rotate the camera around the origin.
Scroll to zoom.
"R" to reset the scene.
"T" to pause and unpause time.
"Y" to step the simulation by a single frame (only works when the simulation is paused).
"ESC" to quit.
```
